Posicionamiento en CSS

Una guía visual con ejemplos prácticos

El posicionamiento en CSS es una herramienta poderosa para controlar la ubicación de los elementos en la página. Existen diferentes modos de posicionamiento, cada uno con su comportamiento específico.

A continuación, exploraremos cada tipo de posicionamiento con ejemplos visuales y código de implementación.

static

Valor por defecto. Los elementos se colocan en su posición natural dentro del flujo del documento.

Referencia
static
Referencia
position: static;
/* No se puede modificar con top, right, bottom, left */
relative

Permite desplazar un elemento respecto a su posición original sin alterar el flujo del documento.

Referencia
relative
Referencia
position: relative;
top: 20px;
left: 30px;
absolute

Posiciona el elemento relativo a su contenedor posicionado más cercano o al viewport si no hay un contenedor posicionado.

Referencia
absolute
Referencia
position: absolute;
top: 20px;
right: 20px;
fixed

El elemento se fija en una posición relativa al viewport y no se mueve al hacer scroll.

fixed

Este elemento está fijo en la esquina inferior derecha de la ventana.

position: fixed;
bottom: 20px;
right: 20px;
sticky

Combina características de relative y fixed. El elemento se comporta como relative hasta que alcanza un umbral, momento en que se "pega" y se comporta como fixed.

Comparativa de Tipos de Posicionamiento

Tipo Relativo a En el flujo del documento Se puede desplazar Uso común
static Flujo normal Sí No Elementos en flujo normal
relative Su posición original Sí (reserva espacio) Sí (top, right, bottom, left) Ajustes menores de posición
absolute Contenedor posicionado más cercano No Sí Elementos superpuestos, menús, tooltips
fixed Viewport No Sí Headers, footers, botones flotantes
sticky Contenedor de desplazamiento Sí (hasta que se "pega") Sí (solo en el eje definido) Headers de tablas, menús de navegación
